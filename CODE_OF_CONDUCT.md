# Code of Conduct

## Our Commitment

MediaKit aims to provide a welcoming and respectful environment for contributors.

## Expected Behavior

- Be respectful
- Be constructive
- Welcome feedback
- Focus on collaboration
- Help improve developer experience

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Toxic behavior

## Project Focus

MediaKit is focused on building a modern modular Android media SDK ecosystem.
