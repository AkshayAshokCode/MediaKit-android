# Versioning Strategy

## Semantic Versioning

MediaKit follows semantic versioning.

```text
MAJOR.MINOR.PATCH
```

## Rules

### MAJOR
Breaking API changes.

### MINOR
Backward-compatible features.

### PATCH
Bug fixes and stability improvements.

## Stability Goals

Before stable releases:
- Public APIs should stabilize
- Bitmap handling should be reliable
- Lifecycle handling should be verified
- Sample app should demonstrate integrations clearly

## Planned Release Flow

- Alpha releases
- Beta releases
- Stable releases
