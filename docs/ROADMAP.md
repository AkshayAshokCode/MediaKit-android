# MediaKit Roadmap

## Current Focus

### Image Picker
- Lifecycle-safe flows
- Camera support
- Error handling
- Crop integration

### Image Cropper
- Matrix-based crop engine
- Crop overlay improvements
- Bitmap safety
- Performance optimization

## Upcoming Improvements

- Compose support
- Publishing automation
- Safer bitmap handling
- Public API cleanup
- Sample app improvements

## Long-Term Vision

MediaKit aims to become a modular Android media SDK ecosystem with independently usable components.
