# MediaKit Android

Modern Android media toolkit built with Kotlin.

## Current Modules

### Image Picker
- Gallery image picking
- Camera image capture
- Lifecycle-aware handling
- Error callbacks
- Crop integration support

### Image Cropper
- Matrix-based crop engine
- Custom crop overlay
- Crop rectangle controls
- State restoration support
- Bitmap export support

## Project Structure

```text
MediaKit-android/
├── ImageCropper/
├── imagepicker/
└── sample-app/
```

## Goals

- Modular Android media SDK ecosystem
- Lightweight dependencies
- Kotlin-first APIs
- Modern Android architecture
- Compose-friendly future support
